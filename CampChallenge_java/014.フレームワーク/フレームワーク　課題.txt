java EE の導入方法（GlassFishの場合）

使うIDEはNetBeans

1.GlassFish4をダウンロード

2.適当な所に解凍

3.NetBeans内のツールの中のサーバーを選択

4.サーバーの追加を選択

5.GlassFish Serverを選択

6.ダウンロードしたGlassFish4直下のGlassFishファイルを選択

7.必要事項を書いて作成

8.後は好きなようにプロジェクトをつくるだけ


Apache Struts　特徴

JSPで、特有のの「＜％％＞」を使う必要がない。
他人にサーバをいじられる脆弱性がある。そのため、Struts2になった。
だが、１、２両方とも被害を出しているため、セキリティにおける脆弱性が目立つフレームワーク


Spring Framework　特徴

変更に強い・拡張性が高い・保守性が高い・再利用性が高い
